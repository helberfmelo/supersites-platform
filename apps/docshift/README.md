# DocShift

Browser-side PDF and document workflows.

## Scope

- Nuxt SSG public app with five locale route sets.
- Free tools:
  - PDF merge;
  - PDF split;
  - PDF rotate;
  - PDF compressor;
  - PDF watermark;
  - page numbers;
  - metadata cleaner;
  - text to PDF.
- One browser-side workflow at a time, with small-file limits.
- Worker validation plans file count, type, size, page ranges and text limits before `pdf-lib` creates the output in the browser.
- Selected files, page text, file names, metadata and generated bytes are not sent to a product API or analytics.

## Commands

```powershell
pnpm test:docshift
pnpm build:docshift
pnpm validate:docshift-preview
pnpm test:e2e:docshift
```
